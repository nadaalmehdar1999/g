Connect Web Page Recogrition with Arduino has Servo Motor


Discription: 
after speech has recongnized, the text will make the servo motor moving left or right.


Languages and Technologies: 
Updating the web Recogrition page in previous task
JavaScipt
Serial API


Environemt: 
Serial API is supported by Chrome only.


How to Use:
firtly, You will reconize your speech and will appear in the textbox. Then, using serial API will send the text to arduino code to controll motor moving.
There are only 2 movements: right and left.
Note: The movement will start after ending the recognition.
